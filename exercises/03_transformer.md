# Transformer 练习

## 1. 未来信息泄漏

构造长度为 4 的 Q/K/V。把最后一个 Value 加 1000，证明前 3 个位置输出不变。若改变，定位 mask 的方向错误。

## 2. MHA 与 GQA

对 `d_model=32, q_heads=8, kv_heads=2, sequence=1024`，比较 MHA 与 GQA 单层 K/V Cache 元素数。说明为何 Query 参数不计入持久缓存。

## 3. KV Cache Oracle

使用同一个注意力模块：

- 路径 A：一次输入完整序列。
- 路径 B：每次输入一个 token 并累积 cache。

逐位置比较输出。先关闭 dropout，并解释训练模式下为什么不应期待逐元素一致。

## 4. RoPE

证明二维旋转保持 L2 范数。再用代码检查不同位置的同一向量，观察点积如何随相对距离变化。

## 5. Tiny GPT

在两个长度为 4 的样本上过拟合模型。记录初始/最终 loss、梯度范数和贪心生成。写出三个“能过拟合却仍然实现错误”的例子。
