# 练习区

练习按“先独立做、再看提示、最后看答案”组织：

第一次学习时先进入 `starter/`：其中五个 Python 文件保留 TODO，并带有可直接运行的最小断言。
默认 `pytest` 不执行 starter，因此“仓库测试通过”不等于“你已经完成练习”。


1. `01_foundations.md`：形状、梯度、Softmax。
2. `03_transformer.md`：因果注意力、KV Cache、GQA。
3. `07_moe.md`：Top-k、容量、负载均衡。
4. `hints.md`：只给思路，不给完整代码。
5. `solutions/selected_solutions.md`：精选答案与验收方法。

不要通过“输出看起来合理”判断代码正确。先写最小反例和数值 oracle，再实现。
