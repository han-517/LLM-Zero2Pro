# 基础练习

## 1. 形状追踪

已知 `x:[2,5,12]`、`W:[12,36]`。计算 `qkv=x@W` 后把最后一维切成三份，再变形成 3 个 head。写出每一步形状，并说明哪一维被矩阵乘法求和。

## 2. 手算链式法则

令 `y = tanh(x*x + 3*x)`，在 `x=2` 时手算 `dy/dx`，再用 `Value` 验证。

## 3. 稳定 Softmax

直接计算 `[1000, 1001, 1002]` 的 Softmax 会发生什么？实现“先减最大值”的版本，并证明平移所有 logits 不改变概率。

## 4. 单 batch 过拟合

创建 4 个二维点和二分类标签，训练一个两层 MLP 直到 loss 低于随机基线的 10%。若失败，依次检查 target、梯度、学习率和激活。
